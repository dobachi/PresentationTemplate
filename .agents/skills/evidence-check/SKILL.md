# エビデンスチェックアシスタント

指定したパスやテキストを元に、参考文献や引用の妥当性を検証する。

## 手順

1. **対象の特定**
   - ファイルパスが指定されている場合、そのファイルを読み込む。
   - テキストが指定されている場合、該当箇所を検索する。
   - 引数なしの場合、カレントディレクトリ配下の文書ファイル（.md, .txt, .tex等）を検索する。

2. **参考文献の抽出**
   - URLリンク（`http://...`, `https://...`）
   - 文献引用（`[1]`, `(Smith, 2020)`）
   - 脚注参照
   - 画像・図表の出典

3. **整合性の検証**
   - リンク先が存在するかを確認する。
   - 引用された数値・情報が原文に存在するか確認する。
   - 引用の使い方が適切か評価する。
   - アクセス不可の場合は手動確認を依頼する。

4. **レポート生成**
   - 問題なし、自動修正可能、人間確認必要、重大な問題に分類する。
   - 修正案を提示する。
   - 承認されれば修正を適用する。

## 注意事項

- 有料コンテンツや認証が必要なページはアクセスできない。その場合は手動確認を依頼する。
- 修正を適用する前に必ず承認を得る。
- 複数の参考文献がある場合は並列でチェックすると効率的。
